Sobre nós

Somos alunos da ETEC e atualmente cursamos Desenvolvimento de Sistemas. Buscamos constantemente adquirir novos conhecimentos, tanto nas aulas quanto por meio de cursos online e estudos independentes. Atualmente estamos aprofundando nossos conhecimentos em desenvolvimento front-end com HTML5, CSS3, JavaScript, PHP e React. Porém também estudamos desenvolvimento back-end com C# e Python, além de conceitos de banco de dados e modelagem de dados.


Tecnologias utilizadas nas aulas

React Native é um framework de código aberto criado pela Meta Platforms, Inc. É usado para desenvolver aplicativos para Android, Android TV, iOS, macOS, tvOS, Web, Windows, e UWP.

Expo é um framework e plataforma de código aberto para React Native que simplifica a criação, construção e implantação de aplicativos nativos para Android, iOS e web, usando JavaScript ou TypeScript.
